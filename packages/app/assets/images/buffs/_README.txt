버프 아이콘을 여기에 넣으세요 (파일명 = BuffKind.key). .webp 또는 .png 모두 인식.

  goldRush   ← 금화 더미 (골드 획득 ×2)
  xpBoost    ← 초록 도감 + 위 화살표 (경험치 ×2)
  frenzy     ← 빨간 불꽃 교차 뿔/블레이드 (공격력·공속)
  gatherer   ← 재료 쏟아지는 주머니 (재료 획득 ×2)
  luckyWind  ← 네잎클로버 + 나비 (곤충 발견율 ×2)

예: goldRush.png (또는 goldRush.webp). 넣고 앱 재실행(또는 hot restart)하면 표시됨.
없으면 이모지로 자동 폴백.
