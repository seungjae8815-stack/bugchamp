# 스토어 등록용 애셋

Play Console → 성장 → 스토어 설정 → 기본 스토어 등록정보 에 올릴 파일들.

| 파일 | 용도 | 규격 | 상태 |
|---|---|---|---|
| `app_icon_512.png` | 앱 아이콘 | 512x512 PNG | ✅ |
| `screenshots/*.png` | 휴대전화 스크린샷 5장 | 1080x1920 (9:16) | ✅ |
| (없음) | 그래픽 이미지(피처) | **1024x500** | ❌ 사장님 |

## 스크린샷 (순서대로 올릴 것)

1. `01_방치채집` — 방치 채집 화면(레벨·재화·서식지·보스)
2. `02_전투매치업` — 팀 편성 + 상생 +10% + 상대 3택1
3. `03_상성휠` — 수동 전투(심해 배경·공방회 휠·기력·10초)
4. `04_채집함` — 곤충 그리드 + 장착 펫
5. `05_곤충상세` — 개체 정보·수련·합성·부위강화

## 만든 방법 (다시 뽑을 때)

폰 실기에서 `adb exec-out screencap` 으로 캡처한 뒤 파이썬으로 가공했다.
원본은 1080x2340(9:19.5)이라 **Play 요구 비율(9:16)에 맞지 않으므로**,
상태바(95px)·네비게이션바(135px)를 잘라내고 비율을 유지한 채 축소해
1080x1920 캔버스 중앙에 배치했다(내용 잘림·왜곡 없음).

원본 캡처는 `raw/`(gitignore)에 남는다.
