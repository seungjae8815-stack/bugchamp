## 소모성 / 비소모성

구글은 등록 시 이를 구분하지 않는다. 앱이 `consume` 하느냐로 갈리고,
그건 코드가 `iap.json` 의 `kind` 를 보고 처리한다. **전부 관리형으로 만들면 된다.**

| 반복 구매 가능(소모성) | 1회 한정(비소모성) |
|---|---|
| `jelly_s`, `jelly_m`, `jelly_l`, `jelly_xl`, `jelly_xxl` | `remove_ads`, `starter_pack`, `idle_pass`, `skin_gold_rhino`, `skin_albino_stag`, `theme_arena` |

## 등록 후 확인

- [ ] 11개 모두 **활성** 상태인지
- [ ] 앱 상점 화면에서 가격이 **콘솔 가격**으로 표시되는지(스토어 연결 확인)
- [ ] '개발 모드' 배너가 사라졌는지
