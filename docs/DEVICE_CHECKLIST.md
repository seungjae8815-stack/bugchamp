### 0-1. ★ 서버권위 검증 빌드 (전투·재화를 서버가 확정)
> 기본 실행(위)은 **로컬 계산**이다 — 전환 중 안전장치라 `GAME_SERVER_URL` 이
> 없으면 서버에 안 붙는다. 서버 경로를 실기에서 밟으려면 URL 을 얹어 실행한다.
> URL 은 비밀이 아니므로 명령에 직접 적는다(supabase 키만 파일에서 주입).

```
cd packages\app
flutter run -d <기기> --dart-define-from-file=supabase.env.json ^
  --dart-define=GAME_SERVER_URL=https://bugchamp-server-867649520275.asia-northeast3.run.app
```
(PowerShell 에서는 줄바꿈 `^` 대신 한 줄로 붙이거나 백틱 `` ` `` 사용)

- [ ] 설정(⚙️) 하단에 **🔐 서버권위** 칩이 뜬다 (이 칩이 없으면 아직 로컬 — 검증 무효)
- [ ] **곤충 획득·강화·수련** 후 앱을 껐다 켜도(=서버 재조회) 수치가 유지된다

- [ ] **야생 수동 전투**: 매 수 진행이 부드러운가 — **서울 리전 왕복 지연**을 체감 확인
      (심리전 리듬이 답답하면 알려주세요 → 응답 프리페치로 완화)
- [ ] **구매**(젤리 등): 결제 후 재화가 서버 기준으로 반영
- [ ] KO 된 곤충으로 바로 재출전 시도 → **부상 회복 타이머**로 편성 거부되는지
