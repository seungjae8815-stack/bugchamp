## 1. 코어 루프

1. 캐릭터가 현재 구역을 **자동으로 오른쪽으로 전진**한다.
2. 앞에 나타나는 **서식지**(나무/꽃/돌/그루터기/버섯…)를 **채집도구로 자동 타격** → 서식지 **HP** 감소.
3. HP 0 → 서식지 **파괴 + 보상 폭발**(골드/재료/경험치, 확률적 **곤충 조우**).
4. 구역의 서식지 N개 클리어 → **보스**(거대 서식지 또는 거대 곤충) 등장 → 처치 시 **대량 보상 + 다음 구역 해금**.
5. 획득 재화로 **캐릭터 능력치 업그레이드**(공격력/공속/이동속도/보상량/치명타) → 클리어 속도 상승.
6. **방치**: 오프라인 동안 현재(클리어 가능한) 구역을 자동 반복 파밍, 재화 누적 **8h 상한**. 복귀 시 정산.
7. 조우한 **곤충은 도감/보관함**에 수집(수집·육성 레이어 유지).

**성장의 축**: 캐릭터 스탯 업그레이드(속도) ↔ 스테이지 진행(벽) ↔ 재화 수급(파밍). 전형적 idle 3축 루프.

## 2. 화면 구성 (횡스크롤)

```
┌───────────────────────────────────────────────┐
│ [HUD] 💰골드  🔷경험치바(Lv)  🧪재료  ⚙설정      │  상단
├───────────────────────────────────────────────┤
│  (패럴랙스 배경: 하늘/원경/중경/지면)            │
│                                                 │
│   🧑‍🌾→(타격모션)     🌳[HP▓▓▓░]  ← 서식지        │  씬(플레이 영역)
│      코인 튀어오름↑↑                             │
│                                                 │
├───────────────────────────────────────────────┤
│  스테이지  1-3   [■■■□□ 서식지 3/5 · 보스까지]   │  하단 진행
│  [⚔공격 Lv12] [⚡공속 Lv8] [👟이속] [💰보상]  ← 업그레이드
└───────────────────────────────────────────────┘
   [ 홈 · 도감 · 업그레이드 · (브리딩) · 상점 ]  탭
```

- **캐릭터**: 화면 좌측 1/3, 걷기→타격→(획득)기뻐 모션 반복.
- **서식지 오브젝트**: 캐릭터 앞에 등장, 머리 위 HP바(UI). 타격 시 흔들림/파편, 파괴 시 폭발.
- **보상 연출**: 코인·재료 아이콘이 튀어올라 상단 카운터로 빨려듦.
- **하단**: 현재 스테이지(지역-구역), 미니 진행바(이 구역 서식지 진행/보스), 스탯 업그레이드 바로가기.

## 3. 시스템 정의

### 3.1 서식지(Habitat)
- 필드: `type`(나무/꽃/돌/그루터기/버섯…), `maxHp`, `dropTable`(파괴 시 골드/재료/경험치/곤충 확률), 등장 지역.
- HP는 구역 깊이에 비례 상승(진행 벽 형성). **밸런스는 JSON**(habitats.json / stages.json).

### 3.2 스테이지(Stage) = 지역-구역
- 표기 `지역-구역` 예: **1-1 ~ 1-10** 후 **2-1**…
- **지역(Region)** = 테마 = **기존 필드 재활용**: 1=참나무 숲, 2=계곡 물가, 3=풀숲 초원, 4=야산 밤숲.
- 각 구역 = 서식지 N개(파괴) + **보스 1**. 보스 처치로 다음 구역 해금.
- 보스 못 깨면 그 구역/직전 구역 반복 파밍(방치 진행).

### 3.3 보스(Boss)
- 거대 서식지(고목/거암) 또는 **거대 곤충**(기존 전설·영웅 곤충 활용: 장수하늘소/장수말벌/물장군/왕사슴벌레).
- 처치 보상 배수(골드·경험치↑), 첫 처치 보너스. 지역 테마별 보스.

### 3.4 캐릭터 & 능력치
- **단일 주인공**(확정). 스탯: `공격력`, `공격속도`, `이동속도`, `보상배율`, `치명타확률/피해`.
- **입력(확정)**: 자동 타격 + **탭 부스트**(탭 시 일정 시간 데미지/공속 증가, 또는 즉시 추가타).
- 업그레이드 = **골드 소모**, 레벨당 비용 상승(지수 곡선) / 효과 상승. 곡선 계수는 **JSON**(upgrades.json).
- **경험치→레벨**: 기본 스탯 소폭 상승(보스에서 대량 획득).

### 3.5 재화 이코노미
| 재화 | 용도 | 주 획득처 |
|---|---|---|
| 💰 골드 (soft) | 능력치 업그레이드 | 서식지 파괴(대량) |
| 🔷 경험치 | 캐릭터 레벨업 | 보스·서식지 |
| 🧪 재료(키틴/미네랄/수액/젤리) | 곤충 부위 강화(기존) | 특정 서식지·곤충 |
| 🍮 젤리 (premium/IAP) | 가챠·편의·광고제거 | IAP/보상형 광고 |
- ❌ 스탯 직접 판매 금지(기존 P2W 방지 원칙 유지).

### 3.6 곤충 수집(유지)
- 서식지 파괴/보스 처치 시 확률적 **곤충 조우** → 도감·보관함(기존 `IndividualBug`).
- 곤충 → 캐릭터 **패시브 버프**(예: 특정 곤충 보유 시 골드+%, 공격+%)로 육성 동기 부여(확정).

### 3.7 방치/오프라인
- 오프라인 동안 **클리어 가능한 최고 구역을 자동 반복** 파밍, 재화 누적 8h 상한. 복귀 시 정산 팝업.

## 4. 필요 애셋 목록 (★ 프롬프트용)

> **시점 주의**: 횡스크롤이라 전부 **측면(side-view)** 시점. 기존 곤충 카드(3/4 하향)와 **별도 세트**.
> **스타일**: `docs/art_prompts.md` 의 "코지 자연주의 카툰" 접미사 유지 + **side view**.
> **투명 배경**: 단색으로 뽑고 rembg 컷아웃. **배경 레이어는 가로 seamless 타일**.

### A. 배경 (지역별 · 패럴랙스 · 가로 타일링)
지역 = 참나무 숲 / 계곡 물가 / 풀숲 초원 / 야산 밤숲(밤).
각 지역마다 레이어 분리:
- **L1 하늘/원경**: 하늘·노을·먼 산 실루엣 (가로 반복)
- **L2 중경**: 나무숲/능선 실루엣 (반투명, 느린 패럴랙스)
- **L3 근경 장식**: 풀·덤불·바위 띠
- **L4 지면 타일**: 바닥(흙/풀/자갈), **가로로 끝없이 이어지는 seamless 타일**
- 사양 예: 각 레이어 2048×1024(또는 폭 넓게), 지면 타일 512×256 반복.

### B. 캐릭터 모션 (측면 · 스프라이트 프레임)
플레이어 1종(도구 든 채집가), 향함 →. 상태별 프레임 시퀀스(스프라이트 시트 or 낱장):
- **idle** 2~4프레임(호흡)
- **walk** 6프레임(걷기 루프)
- **attack** 3~4프레임(도구 휘두르기)
- **cheer** 2프레임(획득 기뻐)
- (선택) **hurt / levelup**
- 사양: 투명, 동일 캔버스(예 512×512), **발끝 앵커 일관**, 프레임 간 정합.
- 도구는 우선 1종(채집망 or 곡괭이). 도구 스킨은 추후.

### C. 서식지 오브젝트 (파괴 가능 · 측면)
지역 테마별 2~4종. 예: 나무 / 꽃덤불 / 바위 / 그루터기 / 버섯 / 수풀 / (계곡) 이끼바위·통나무 / (밤) 고목·발광버섯.
각 오브젝트 **상태 세트**:
- **정상** → **손상(균열, ~50%)** → **파괴 직전(~20%)** → **파편 조각 3~5개**(폭발 연출용)
- 사양: 투명, **지면 앵커**, 세로형. HP바는 UI로 얹으므로 그림 불필요.

### D. 보스 (구역 끝 · 측면 · 대형)
- 거대 서식지(고목/거암) 또는 거대 곤충(기존 곤충 측면·대형 리디자인).
- 모션: **idle / hit(피격 반짝) / death(파괴)** 최소 3상태.
- 지역별 1개 이상.

### E. 재화 · 아이템 · 곤충
- 아이콘(정사각, 투명): 💰골드코인, 🔷경험치 오브, 재료 4종(키틴/미네랄/수액결정/젤리), 🍮젤리.
- **드롭 스프라이트**: 파괴 시 튀어오르는 소형 코인/재료(1~2프레임).
- **곤충 20종**: 기존 3/4 카드 그대로(도감·조우 카드용).

### F. 이펙트 (스프라이트 시트)
- **타격 임팩트**(스파크/먼지 튐), **파괴 폭발**(빛+파편), **코인 트레일**(튀는 궤적),
  **치명타**(강조 숫자 배경/섬광), **레벨업 오라**, **보스 처치 섬광**.

### G. UI
- HUD 프레임/패널, **스탯 업그레이드 아이콘**(⚔공격·⚡공속·👟이속·💰보상·✨치명타), **스테이지 배지**(1-1 표기 틀),
  **진행바 프레임**, 버튼 세트, 업그레이드 패널 배경, 등급 프레임(기존).

### H. 기타
- 앱 아이콘, 스플래시, 도구 아이콘, (선택) 스킬/버프 아이콘.

## 5. 코드/기존 자산 관점

- **유지**: `core_models`(Species·IndividualBug·Material·enums·정규분포 롤), JSON-데이터 규칙, 결정론, Hive 세이브(마이그레이션), 다국어(ARB), `core_gathering`의 오프라인 정산 아이디어(→스테이지 파밍으로 재사용), `core_battle`(빈 뼈대→서식지 데미지/전투 계산으로 활용).
- **신규 모델(예정)**: `Habitat`, `Stage/Region`, `Boss`, `CharacterStats`, `Upgrade`, `Currency(gold/xp)`.
- **신규 데이터 JSON**: `stages.json`, `habitats.json`, `upgrades.json`, `bosses.json`.
- **UI 재작성**: 홈 = 횡스크롤 플레이 화면. 트랩 설치 UI는 제거/대체.
- **곤충 수집·도감·브리딩·PvP**: 유지 범위는 Q1/Q2 결정에 따름.

## 6. 확정 사항 (2026-07-06)
- **곤충 레이어**: 유지 + **버프 연결**(보유 곤충 → 캐릭터 패시브 버프: 골드+%, 공격+% 등). PvP 는 Phase 4.
- **캐릭터**: **단일 주인공** 1명 성장(모션 1세트).
- **채집 입력**: **자동 타격(방치) + 탭 부스트**(탭 시 추가 데미지/속도).

## 7. 애셋 생성 우선순위 (임팩트순)
1. **스타일 키 1장** — 캐릭터 idle 측면 1장으로 톤 확정(→ 이후 `--sref` 로 통일)
2. **배경 1지역(참나무 숲)** 패럴랙스 4레이어 + 지면 타일
3. **서식지 3종**(나무/바위/꽃덤불) 상태 세트(정상→균열→파괴직전) + **파편**
4. **캐릭터 walk / attack** 프레임
5. **재화 아이콘 + 드롭 스프라이트**, **타격/파괴 이펙트**
6. **1지역 보스 1종**
7. UI(업그레이드 아이콘·스테이지 배지·진행바)

→ **1~5 만 있어도 "스테이지 1-1 플레이"가 성립**한다. 나머지는 지역 확장 시 추가.
