# 수익화 기획 (IAP + 광고)

> 확정: 2026-07-20. 수치는 전부 `packages/app/assets/data/iap.json` (§6 — 코드 하드코딩 금지).
> 원칙: **스탯 직접 판매 금지**(§2.6). 판매 범위는 *편의 + 진행속도*까지 (사장님 확정).

---

## 1. 판매 경계 (P2W 정책)

| 판매 가능 | 판매 금지 |
|---|---|
| 광고 제거, 시간 단축(즉시완료), 슬롯 확장 | **곤충 스탯/포텐셜 직접 판매** |
| 젤리(프리미엄 화폐), 골드·재료 묶음 | 전투 승률을 직접 사는 아이템 |
| 코스메틱(스킨·테마) — 스탯 영향 0 | 상대보다 강해지는 전용 곤충 |
| 진행속도(오프라인 상한, 방치 획득량) | |

> 진행속도 상품은 무과금 격차의 근원이므로 **수치를 JSON으로 상시 조절**한다.

## 2. 상품 카탈로그

### ① 광고 제거 패스 `remove_ads` — ₩7,700 (비소모성)
- 강제 광고 제거. **보상형 광고는 유지**(원할 때 시청 → 보상).
- 매일 접속 시 젤리 10 자동 지급.

### ② 스타터 패키지 `starter_pack` — ₩5,500 (계정당 1회)
- 젤리 300 + 골드 200,000 + 부화기 슬롯 +1 + 재료(키틴/미네랄/수액) 각 500.
- 개별 가치 대비 약 4배 — 첫 결제 전환용.

### ③ 젤리 팩 (소모성)
| 상품 id | 젤리 | 보너스 | 가격 |
|---|---|---|---|
| `jelly_s` | 60 | — | ₩1,200 |
| `jelly_m` | 330 | +10% | ₩5,500 |
| `jelly_l` | 700 | +17% | ₩11,000 |
| `jelly_xl` | 1,500 | +25% | ₩29,000 |
| `jelly_xxl` | 3,300 | +37% | ₩59,000 |

### ④ 스킨·코스메틱 (비소모성, 스탯 영향 0)
- `skin_gold_rhino` 황금 장수풍뎅이 ₩3,300
- `skin_albino_stag` 알비노 사슴벌레 ₩3,300
- `theme_arena` 아레나 테마(전투 장소 배경 스킨) ₩2,200

**적용 방식(2026-07-20 구현)**: 새 아트 없이 **ColorFilter** 로 처리한다.
`iap.json` 의 `skins` 가 "어느 종 접두사에 어떤 효과인지"를 정의(§6) —
`gold_rhino`→`rhino_`(금색 틴트), `albino_stag`→`stag_`(탈색+증백),
`arena_theme`→ 전투 배경 색보정. 보유하면 **내 곤충 전 화면에 자동 적용**
(장착 UI 없음). **상대 곤충에는 적용하지 않는다** — 내 스킨이지 상대 것이 아니므로.
새 아트가 생기면 같은 자리에서 필터 대신 이미지로 갈아끼우면 된다.

### ⑤ 곤충학자 패스 `idle_pass` — ₩9,900 / 30일 (기간제)
- 오프라인 상한 **8h → 12h**
- 매일 젤리 30
- 광고 제거 포함
- 방치 골드 **+20%**

## 3. 기술 설계

- **패키지**: `in_app_purchase`(Flutter 공식). 소모성(젤리) / 비소모성(광고제거·스킨) / 기간제(패스).
- **데이터**: `assets/data/iap.json` → `IapConfig`(core_run, 순수 Dart).
- **세이브 v16**: `adsRemoved`, `starterBought`, `ownedSkins[]`, `passExpiresAt`.
- **복원(restore)**: 비소모성 필수(스토어 심사 항목).
- **영수증 검증**: 초기 로컬 → 이후 Supabase Edge Function 서버 검증.
- **화면**: 상점 탭 `[제작 | 상점]` 2섹션.

## 4. 외부 작업 (사장님)
